пузырьковый метод поиска наименьшего
a, b, c = map(int, input().split())
if a > b:
    a, b = b, a
if a > c:
    a, c = c, a
print(a)

еще способ вывода наименьшего

if a > b:
    a = b
if a > c:
    a = c
